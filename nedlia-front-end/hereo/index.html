Act I — The Editing Bay

Nedlia Plugin

For production studios and video editors

A native plugin for Final Cut Pro, DaVinci Resolve, and LumaFusion. Editors mark product placements directly on the timeline — the same way they add closed captions. Each placement carries the product name, brand, time range, and scene metadata.

  • ✓ Time-range markers with product metadata on the editing timeline
  • ✓ Generates a .nedlia placement file embedded alongside the video
  • ✓ Syncs to Nedlia servers for real-time budget validation downstream
  • ✓ Scene-level control panel for brand associations and product properties

Act II — The Streaming Platform

Nedlia SDK & API

For streaming platforms and video players

A lightweight SDK for any streaming player. When playback reaches a tagged scene, the SDK queries Nedlia to validate the marketing budget. If active, a shop icon appears. The viewer clicks. A flyout shows the product. Viewers can disable entirely — their screen, their choice.

  • ✓ Shop icon appears only when the campaign budget is validated live
  • ✓ Flyout with product details, brand info, and direct purchase link
  • ✓ Viewer opt-out — the feature respects user preference completely
  • ✓ Tracks impressions and click conversions for campaign analytics
📡
Integration — Drop-in SDK
// Add Nedlia to any streaming player
import { NedliaSDK } from '@nedlia/player-sdk'

const nedlia = new NedliaSDK({
  orgId: 'warner-bros-2025',
  agentId: 'morning-scene-final',
  theme: 'dark'
})

// Attach to your video element
nedlia.attach(document.querySelector('video'))

// Shop icons appear automatically when
// tagged scenes play. Budget validation
// happens server-side via Nedlia API.
